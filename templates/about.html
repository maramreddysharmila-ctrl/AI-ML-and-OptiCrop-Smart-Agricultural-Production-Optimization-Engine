About OptiCrop

A Machine Learning-powered agricultural recommendation system for smart farming decisions.

Project Objective

OptiCrop is a Machine Learning-powered agricultural recommendation system that predicts the most suitable crop based on seven key parameters:

  • Nitrogen (N) – soil nitrogen content
  • Phosphorous (P) – soil phosphorous content
  • Potassium (K) – soil potassium content
  • Temperature – ambient temperature in °C
  • Humidity – relative humidity percentage
  • pH – soil pH value
  • Rainfall – annual rainfall in mm

The system helps farmers, agricultural researchers, agribusiness companies, and policymakers make intelligent farming decisions to maximize yield and optimize resource usage.

Target Audience

  • Agricultural Researchers
  • Agribusiness Companies
  • Policymakers

Machine Learning Models Compared
Decision Tree

Tree-based model that splits data by feature thresholds. Easy to interpret but prone to overfitting.

Random Forest

Ensemble of decision trees. Best accuracy in this project. Robust to overfitting.

Logistic Regression

Linear model for classification. Fast and provides probability estimates.

K-Nearest Neighbors

Instance-based learner that classifies by majority vote of nearest neighbors.

K-Means Clustering

Unsupervised clustering used for exploratory analysis of crop groupings.

Best Model

Random Forest was automatically selected with the highest accuracy of 87.3%.
